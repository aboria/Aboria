Description

LinearTransform public construct/copy/destruct

  1. LinearTransform(const T & point_transform);

    takes a user-defined transform point_transform that is assumed to be linear. point_transform only has to defined an operator()(const Vector<double,D>&) function

LinearTransform public member functions

  1. const bool_d & get_eigen_vertices();

    the transform stores the bounding box vertex that is maximally extended from the centre by the transform, this getter returns this vertex 

  2. double_d operator()(const double_d & v) const;

    transforms the point the the new coordinate system, returning the result 

  3. Vector< double, D > operator()(const bbox< D > & b) const;

    transforms the box to the new coordinate system. Returns the side length of the axis-aligned box that bounds the transformed box.
