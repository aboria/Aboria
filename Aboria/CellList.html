Description

This class implements neighbourhood searching using a bucket search, or cell list algorithm. The domain is first divided up into a regular grid of constant size "buckets".

After the buckets are created, a set of 3D points can be assigned to their respective buckets. After this, neighbourhood queries around a given point can be performed be looking at all the points in the same bucket or surrounding buckets of the given point.
