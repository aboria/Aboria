Description

This class implements neighbourhood searching using a cell list. That is, the domain is divided up into a regular grid of constant size "buckets", and particles are assigned to their containing bucket.

The main difference between this class and CellList is that the particle set is reordered according to which cell each particle belongs to. This correlates memory locality with spatial locality, and improves cache access if you are often looping over neighbouring particles. It also mean that particles within a given bucket are sequential in memory.
